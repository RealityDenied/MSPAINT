# MSPAINT
This is a MS Paint clone , with features like Pencil Tool , Eraser Tool , Paint bucket Tool , Line Tool , Rectangle Tool
made using OPENGL GLUT , just setup your OPENGL with GLUT and paste this CPP file provided.
